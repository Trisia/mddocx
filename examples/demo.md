# 第一章 深度学习实验报告

## 1.1 研究背景

近年来，深度学习技术在计算机视觉领域取得了突破性进展。卷积神经网络（CNN）作为核心架构之一，在图像分类、目标检测等任务中表现出色。

## 1.2 实验方法

实验采用 CIFAR-10 数据集，随机选取每类 100 张图片作为训练集。

![模型架构](https://www.python.org/static/img/python-logo.png)

表 超参数配置

| 参数 | 取值 |
|------|------|
| 学习率 | 0.001 |
| 批量大小 | 32 |
| 训练轮次 | 100 |
| 优化器 | Adam |

## 1.3 实验结果

模型在测试集上取得了 94.2% 的 Top-1 准确率，验证了深度学习方法在小样本场景下的有效性。
